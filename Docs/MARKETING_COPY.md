### FAQ Section

**Q: Will Voco run commands without my permission?**
A: Never. Every file write and terminal command requires your explicit approval in the Review Deck. Nothing executes without your "yes."

**Q: Is my voice data stored?**
A: No. Raw audio is streamed for transcription and immediately discarded. Transcripts are retained for 30 days, then permanently deleted.

**Q: Does my code leave my machine?**
A: Your code never leaves your machine. Voco runs locally. Only your voice transcript is sent to Claude for reasoning — never your source files.

**Q: What AI model does it use?**
A: Claude Sonnet by Anthropic — the same model powering Claude Code. You bring your own API key or use our included credits on the free tier.

**Q: Does it work with my IDE?**
A: Yes. Voco runs as a standalone desktop app alongside any editor — VS Code, Cursor, Windsurf, IntelliJ, or even vim. It also exposes an MCP server for direct IDE integration.

**Q: What happened to GitHub Copilot Voice?**
A: GitHub discontinued Copilot Voice in April 2024. Voco is the voice-native coding agent that picks up where they left off — with AI reasoning, local execution, and human-in-the-loop safety that Copilot Voice never had.

**Q: I have RSI. Can Voco replace my keyboard?**
A: Voco significantly reduces typing for AI-assisted coding workflows. Combined with tools like Talon for general computer control, many developers with RSI use Voco as their primary coding interface.
