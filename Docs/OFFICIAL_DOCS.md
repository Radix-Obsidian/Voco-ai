# Voco Documentation

## Getting Started
1. Download Voco for your platform
2. Install and launch
3. Grant microphone permissions
4. Say "Hey Voco" to begin

## Voice Commands
- "Search for [query]" — Search codebase
- "Create [feature]" — Generate code
- "Run [command]" — Execute terminal
- "Stop" — Interrupt AI mid-sentence

## Zero-Trust Security
- All code execution requires approval
- Filesystem access is sandboxed
- Your code never leaves your machine

## Settings & API Keys
Navigate to Settings → API Keys to add:
- Deepgram (speech-to-text)
- Cartesia (text-to-speech)
- Anthropic/OpenAI (AI reasoning)

## Pricing Tiers
- **Listener** (Free): 500 voice turns/month
- **Orchestrator** ($39/mo): Unlimited voice
- **Architect** ($149/mo): Priority + custom skills

## FAQ
Q: Does Voco store my code?
A: No. All code execution is local.

Q: What languages does Voco support?
A: All languages. Voco reads and writes any text file.

## Troubleshooting
- Audio issues: Check microphone permissions
- Connection issues: Verify API keys in Settings
- Crashes: Report bugs on GitHub Issues
